## 미션 – 숫자 야구⚾

### 구현 기능 목록

- [x] 게임 시작 기능
- [x] 컴퓨터의 랜덤 숫자 생성
    - [x] 3자리 숫자가 모두 다른 수로 구성되는지 확인
- [x] 숫자 입력 기능
    - [ ] 유저로부터 3자리 숫자 입력
    - [ ] 자릿수 확인(예외 발생)
    - [ ] 입력된 값이 숫자인지 확인(예외 발생)
    - [ ] 0이 입력되었느지 확인(예외 발생)
    - [ ] 중복된 수가 있는지 확인(예외 발생)
- [ ] 정답과 입력값 비교
    - [ ] 스트라이크
    - [ ] 볼
    - [ ] 낫싱
- [ ] 종료 및 재시작
    - [ ] 1 혹은 2를 입력했는지 확인(예외 발생)
    - [ ] 1(재시작)시 위의 과정 반복
